# OBSERVABLE-CATEGORY-SKETCH

Sketch note testing whether the F-obstruction becomes cleaner when stated
as a non-existence statement between small categories of named observables,
rather than between the log side and circle side as whole spaces.

Verdict in advance: **partial, not primary**. The observable-category frame is
useful for typing and for preventing false identifications between
observables, but the actual no-go still concentrates in the closure-depth
mismatch: log-side native closure is flat inside `Aff^+(R)`, while the circle
side has an unbounded cyclotomic/Chebyshev ladder. Rate and arithmetic
asymmetry are valuable diagnostics, not yet independent enough to replace
the closure-depth theorem.

---

## Objects

This is a small named-observable sketch, not an abstract category-building
exercise. Objects are the observables already used in the repo. Morphisms
are generated by native operations on each side.

### `C_log`

Objects:

| Object | Meaning | Home |
|---|---|---|
| `L` | Mitchell affine pseudo-logarithm `L(x) = E + m` | `memos/LANDFALL-EXPORT.md` |
| `Aff` | native affine closure object, maps `x -> ax + b` | `memos/LANDFALL-EXPORT.md` |
| `lambda` | clock-change `lambda(m) = log_2(1+m)` | `memos/LANDFALL-EXPORT.md` |
| `epsilon` | residue `epsilon(m) = log_2(1+m) - m` | `memos/LANDFALL-EXPORT.md` |
| `epsilon_dyadic` | interior dyadic samples of `epsilon` | `memos/LANDFALL-EXPORT.md` |

Generating morphisms:

- affine composition inside `Aff`;
- pre-composition and post-composition by rational affine maps;
- the representation relation `log_2 x = E + lambda(m)`;
- the residue relation `epsilon = lambda - id`.

The key negative fact is that finite affine composition never generates
`lambda` exactly, hence never generates `epsilon` exactly.

Generating morphisms:

- Chebyshev composition, `trace(R_n^m) = 2 T_m(cos(2pi/n))`;
- cyclotomic field generation and inclusions;
- Galois actions on `Q(zeta_n + zeta_n^-1)`;

The key positive fact is that finite circle-side operations generate an
unbounded family of fields of degree `phi(n)/2`.

## Invariants

Attach three computable invariants to objects when the invariant is
meaningful. Not every invariant applies to every object.

### 1. Closure depth `d`

Convention used here:

```text
d(X) = native closure depth needed to generate X exactly.
```

If an object is not reachable by finite native closure, write `unreached`.
If the object is analytic/variational rather than algebraic-closure-native,
write `analytic`.

| Object | `d` |
|---|---:|
| `L` | free at representation layer |
| `Aff` | flat finite closure |
| `lambda` | unreached by finite `Aff` |
| `epsilon` | unreached by finite `Aff` |
| `epsilon_dyadic` | unreached; pointwise transcendental at interior dyadics |
| `trace_n` | `phi(n)/2` |
| `tau_n` | `phi(n)/2` off Niven zeros; zero on `{1,2,3,4,6}` |
| `K_n` | `phi(n)/2` |
| `X_n` | `phi(n)/2` by row-field collapse |
| `Delta_n` | analytic / Parseval |
| `A_below` | analytic / strip Fourier |
| `Y_n` | analytic / strip Fourier |
| `H_half` | row-derived; depends on `X_n` arithmetic |

This is the load-bearing invariant.

### 2. Rate `r`

Convention:

```text
r(X) = leading asymptotic rate in the object's native small parameter.
```

| Object | `r` |
|---|---|
| `epsilon` | Fourier coefficients decay `O(q^-2)` in Fourier index `q` from the binade seam; pointwise values do not vanish uniformly |
| `lambda` | no vanishing-rate object; clock-change is non-affine |
| `tau_n` tail | `-4pi^2/n^2 + O(n^-4)` |
| `Delta_n` | `4pi^4/(3n^2) + O(n^-4)` |
| `A_below` | `pi^2/(6n^2) + O(n^-4)` |
| `Y_n` nonzero modes | `d_k = (-1)^k/(k^2 n^2) + O(n^-4)` |
| `H_half` positive guide floor | `pi^2/(4n^2) + O(n^-4)` on `n = 3 mod 6` |

Rate is useful but not decisive. Several objects share `1/n^2` while being
different kinds of observables: tail value, Parseval gap, strip DC mode, and
arithmetic guide distance.

### 3. Arithmetic asymmetry `a`

Use one concrete test:

```text
a(X) = does X distinguish the +1/2 and -1/2 rational guides
       by different approach rates?
```

Values:

| Object | `a` |
|---|---|
| `X_n` / `H_half` | asymmetric: `+1/2` has a `1/n^2` floor on `n = 3 mod 6`, `-1/2` has only a `1/n` envelope |
| `epsilon_dyadic` | no known analogue; Landfall's pointwise dyadic statement is uniform across interior dyadics |
| `tau_n` | not a guide-line observable |
| `Delta_n` | not a guide-line observable |
| `A_below` / `Y_n` | not a guide-line observable |

This invariant is a good observable-selection test. It is not yet a
category-level obstruction except when a proposed functor tries to match
log-side uniform dyadic behavior to the outside-out row's rational-guide
asymmetry.

## Candidate Pairings

### `Aff -> K_n`

This is the central closure-generator pairing.

| Invariant | Log side | Circle side | Result |
|---|---|---|---|
| `d` | flat finite affine closure | unbounded `phi(n)/2` ladder | fails |
| `r` | not a rate object | not primarily a rate object | not decisive |
| `a` | not a guide object | not a guide object | not decisive |

This is the crisp no-go. A functor preserving native closure depth has
nowhere to send the unbounded circle ladder on the log side.

### `L -> trace_n`

| Invariant | Log side | Circle side | Result |
|---|---|---|---|
| `d` | affine/free | `phi(n)/2` | fails |
| `r` | no vanishing-rate object | `trace_n -> 2` at `1/n^2` after subtracting 2 | weak rhyme only |
| `a` | not applicable | not applicable | not decisive |

This is another closure-depth failure. `L` is a free representation readout;
`trace_n` is a generator of unbounded cyclotomic fields.

### `epsilon -> tau_n`

| Invariant | Log side | Circle side | Result |
|---|---|---|---|
| `d` | unreached by finite `Aff` | algebraic of unbounded degree as a family | mismatch in kind |
| `r` | Fourier coefficient decay `O(q^-2)` | tail value `O(n^-2)` | rate rhyme, not equality |
| `a` | no current guide asymmetry | not a guide object | not decisive |

This pairing is suggestive but not theorem-clean. It is the right comparison
for residue rhetoric, but the invariants show why rhetoric is not enough:
the two `1/n^2` appearances live in different registers.

### `epsilon_dyadic -> X_n / H_half`

| Invariant | Log side | Circle side | Result |
|---|---|---|---|
| `d` | pointwise transcendence / unreached by `Aff` | row field `phi(n)/2` | mismatch in kind |
| `r` | no uniform value-decay statement | `+1/2` floor `1/n^2`, `-1/2` envelope `1/n` | fails if rate-asymmetry must preserve |
| `a` | uniform interior dyadic behavior | rational-guide asymmetry | fails |

This is the most useful role for the asymmetry invariant. It does not prove
the main closure mismatch, but it prevents a too-coarse functor from
averaging away the outside-out row's arithmetic structure.

### `Delta_n` or `A_below -> epsilon`

| Invariant | Circle side | Log side | Result |
|---|---|---|---|
| `d` | analytic / Fourier, not cyclotomic-depth primary | unreached by affine closure | not comparable |
| `r` | `1/n^2` Archimedean | `O(q^-2)` Fourier tail for `epsilon` | rate rhyme |
| `a` | none | none | not decisive |

These are good Archimedean-signature comparisons and poor closure-functor
comparisons. They belong in the quantitative companion story, not as the
load-bearing no-go.

## Reframed Theorem Target

A possible observable-category theorem would be:

> There is no functor `F : C_log -> C_circle` that preserves closure depth
> `d`, rate `r`, and arithmetic asymmetry `a` on the named generators above.

The proof immediately localizes at `Aff -> K_n`: preserving `d` would require
a flat affine closure object to map to, or receive from, an unbounded
`phi(n)/2` ladder. No such object exists in `C_log`.

Rate and asymmetry do not strengthen that proof at the same level:

- `r` mostly records rhymes. It explains why these observables belong in the
  same program, but it rarely obstructs a map by itself.
- `a` is genuinely obstructive only for row/near-half comparisons. It is a
  good falsifier for over-compressed observables, not the global no-go.

So the three-invariant theorem collapses, for the decisive pairing, to the
single-invariant closure-depth theorem.
