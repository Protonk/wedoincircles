# NATIVE-F-MINIMAL-DEFINITION

Minimal definition of a native candidate functor for the closure-mismatch
theorem.

Purpose: separate the load-bearing theorem axioms from useful diagnostics.
The theorem should not need to preserve every observable feature in the
repo. It only needs enough structure to force the closure-depth
contradiction.

This memo promotes the "partial, not primary" verdict of
[memos/OBSERVABLE-CATEGORY-SKETCH.md](memos/OBSERVABLE-CATEGORY-SKETCH.md)
into a terse definition.

---

## Small Categories

Use the small named-observable categories from
[memos/OBSERVABLE-CATEGORY-SKETCH.md](memos/OBSERVABLE-CATEGORY-SKETCH.md):

- `C_log`: generated by log-side representation observables such as
  `L(x)=E+m`, `Aff`, `lambda(m)=log_2(1+m)`, and
  `epsilon(m)=log_2(1+m)-m`.
- `C_circle`: generated by circle-side observables such as
  `trace_n=2cos(2pi/n)`, `tau_n`, the trace field
  `K_n=Q(cos(2pi/n))`, the outside-out row `X_n`, Hurwitz gap `Delta_n`,
  and strip tissue `Y_n`.

The precise object list can grow, but the theorem target uses only the
closure-generator objects:

```text
Aff               on the log side,
{K_n}_{n>=3}      on the circle side, with [K_n:Q] = phi(n)/2.
```

## Native Functor Axioms

A functor `F : C_log -> C_circle` is **native** for the program if it
satisfies the following four axioms.

### A1. Basic Functoriality

`F` is a functor:

```text
F(id_X) = id_{F(X)},       F(g o f) = F(g) o F(f).
```

This axiom only prevents relation-breaking. It is not the source of the
obstruction.

### A2. Closure-Generator Preservation

`F` maps native closure-generator data to native closure-generator data and
preserves the closure-depth filtration attached to those generators.

Concretely, the affine closure object `Aff` on the log side may not be sent
to an arbitrary circle observable; it must land in the circle-side closure
apparatus generated by Chebyshev / cyclotomic operations. Conversely, the
circle-side ladder `{K_n}` is a required witness family: a functor that
ignores it is not an `F` for this program.

### A3. Native-Operation Preservation

`F` sends native generating morphisms to native generating morphisms.

Log side:

```text
x -> ax + b,        composition inside Aff^+(R).
```

Circle side:

```text
trace(R_n^m) = 2 T_m(cos(2pi/n)),
cyclotomic field generation and Galois action.
```

If no correspondence can be specified between these generator classes, the
candidate fails A3 before any invariant calculation begins.

### A4. Parameter-Family Correspondence

`F` must carry the relevant parameter families to relevant parameter
families, not just isolated examples.

For the closure witness this means: the image must meet the trace-field
ladder `{K_n}` along an unbounded/cofinal set of `n`. Mapping one log object
to one small crystallographic case is not a native correspondence; it dodges
the witness family.

---

## No-Go Theorem

**Theorem target.** No functor `F : C_log -> C_circle` satisfies A1-A4.

**Proof sketch.**

The circle side contains the required closure-generator family

```text
{K_n}_{n>=3},       [K_n : Q] = phi(n)/2,
```

whose closure depth is unbounded as `n` varies.

By A2 and A4, any native `F` must account for this ladder as the image of a
log-side native closure family. By A3, such a log-side family is generated
by finite composition inside `Aff^+(R)`. But affine composition is flat:

```text
(x -> ax+b) o (x -> cx+d) = x -> (ac)x + (ad+b).
```

So iterating native log operations never leaves the two-parameter affine
form. There is no log-side native closure ladder with unbounded depth.

Contradiction.

The witness localizing the contradiction is `n = 7`: the first cubic trace
field `Q(cos(2pi/7))` and the first non-constructible half-angle node.
The asymptotic form of the contradiction is the unbounded ladder
`phi(n)/2`.

## Diagnostic Preservation Properties

The following properties are useful, but they are not part of the minimal
theorem unless a future argument makes them load-bearing.

| Property | Status | Reason |
|---|---|---|
| Rate-class preservation | Diagnostic | `1/n^2` rhymes across `tau`, `Delta_n`, `A_below`, and near-half gaps, but rate alone does not force closure mismatch |
| Observable-type preservation | Diagnostic, partly A3-shaped | Residue, gap, strip tissue, and row observables should not be conflated; A3 already blocks some type errors |
| Arithmetic-asymmetry preservation | Diagnostic | The `+1/2` vs `-1/2` test catches over-compressed row observables, but does not obstruct `Aff -> K_n` |
| Rate-constant preservation | Diagnostic | Constants such as `4pi^4/3` identify precise observables, but are not needed for the closure-depth no-go |
| Nontriviality / nonzero preservation | Needed only for analytic-only pairings | Prevents trivial functors when closure generators are not in scope |

### Rate-Class Preservation

Example: `epsilon` has Fourier coefficients with `O(q^-2)` decay in the
Fourier index `q`, while `tau_n`, `Delta_n`, and `A_below(n)` have
`1/n^2` Archimedean rates in polygon order `n`.

This is a real quantitative rhyme, but not a proof of functoriality or
non-functoriality. It belongs in
[memos/ARCHIMEDEAN-SIGNATURE.md](memos/ARCHIMEDEAN-SIGNATURE.md), not in the
minimal no-go theorem.

### Observable-Type Preservation

A map sending a residue to a gap, or a strip tissue to a trace field, should
be treated with suspicion unless it also sends the generating morphisms
correctly. Observable type is therefore a typing discipline. It becomes
theorem-load-bearing only when encoded through A3.

### Arithmetic-Asymmetry Preservation

The outside-out row distinguishes the rational guides:

```text
+1/2: 1/n^2 floor on n = 3 mod 6,
-1/2: 1/n envelope.
```

Landfall's pointwise dyadic transcendence statement is uniform across
interior dyadics. If a candidate functor tries to match those two regimes,
the asymmetry probe is a strong falsifier. It is still diagnostic because
the main closure-generator contradiction does not need it.

### Nontriviality for Analytic-Only Pairings

If a proposed `F` only discusses analytic observables such as `Delta_n` and
`A_below`, closure depth may be "not comparable" rather than mismatched. In
that restricted setting, a trivial functor sending everything to zero is not
ruled out by A1 alone.

That is not a defect of the minimal theorem. It means analytic-only
correspondences require extra preservation axioms: rate, nonzero class,
observable type, or a named Fourier coefficient system. Those axioms are
not needed for the closure-generator theorem.

## What Becomes Theorem Language

Use the following as the theorem-facing definition:

| Theorem axiom | Justification |
|---|---|
| A1 basic functoriality | Required for any functorial claim |
| A2 closure-generator preservation | Forces the candidate to touch the actual witness ladder |
| A3 native-operation preservation | Makes "native" mean operation-native, not arbitrary matching |
| A4 parameter-family correspondence | Prevents finite cherry-picking and forces unbounded `n` |

Keep the following as diagnostic language:

| Diagnostic property | Use |
|---|---|
| Rate class | Quantitative companion; Archimedean signature |
| Observable type | Typing hygiene across residues, gaps, rows, and strip tissues |
| Arithmetic asymmetry | Observable-selection/falsifier for row-level maps |
| Rate constants | Identification of exact observables, not closure no-go |

## Promotion Criterion

Promote A1-A4 and the no-go theorem to the headline F-theorem location when
the proof sketch above is written out with the exact closure-depth
definitions:

```text
depth_log(Aff^+(R)) is flat;
depth_circle(K_n) = [K_n:Q] = phi(n)/2 is unbounded.
```

Do not promote rate or asymmetry as theorem axioms unless a future candidate
functor avoids the closure-generator contradiction and is only killed by one
of those diagnostics.
